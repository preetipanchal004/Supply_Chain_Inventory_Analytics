# Walmart Supply Chain & Inventory Analytics

## Project Overview

This project analyzes Walmart supply chain and inventory data using Excel, MySQL, Python, and Power BI.

## Tools Used

- Excel
- MySQL
- Python
- Power BI

## Dataset

- Records: 91,250
- Columns: 16

## Dashboard Features

- Total Revenue
- Total Profit
- Total Units Sold
- Average Inventory
- Revenue by Region
- Revenue by Warehouse
- Supplier Performance
- Promotion Analysis
- Stockout Analysis
